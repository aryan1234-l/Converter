PROJECT TITLE: Value Converter
PURPOSE OF PROJECT: This program will convert numeric values to different value systems.
like There are 4 number value systems namely- 
* Binary
* Decimal
* Hexadecimal
* Octa decimal
The numbers written by us from 0 to 9 have different values in other systems. Like in 
Decimal number system the number 10 is 10 but in binary it's 1010 and in hexadecimal it's
A and in octa decimal it's- 12.
